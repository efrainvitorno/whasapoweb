🚀 Especificaciones Técnicas del CRM Whasao Dev 🛠️ 🚀💼✨

  1. Whasapo Dev v 3.0.0

    Actualizada Enero del 2024

    NOTA: 🚀 La nueva versión de CRM WHASAPO-DEV requiere Node 16.x. ¡Actualizada para seguir disfrutando de las mejoras! 🌟

    Un sistema de tickets muy simple basado en mensajes de WhatsApp. 📝📱

    El backend utiliza whatsapp-Baileys para recibir y enviar mensajes de WhatsApp, crear tickets a partir de ellos y almacenar todo en una base de datos PostgreSQL. 🔄📊

    El frontend es una aplicación de chat multiusuario completa, creada con react-create-app y Material UI, que se comunica con el backend mediante una API REST y Websockets. Te permite interactuar con contactos, tickets, enviar y recibir mensajes de WhatsApp. 📱💬

  2. ¿como funciona?💡

    Actulizado enero del 2024

    ✨ Cómo funciona:

    Cada vez que recibes un nuevo mensaje en un WhatsApp asociado, se genera un nuevo Ticket en estado "🕰️ Esperando". Este ticket se coloca en una cola en la página de Tickets, donde puedes asignártelo al aceptarlo, cambiándolo a estado "👩‍💻 Atendiendo". Luego, respondes al mensaje y, una vez resuelto, modificas el estado a " Resuelto".

    Los mensajes siguientes del mismo contacto estarán relacionados con el primer ticket abierto o pendiente.

    Si el contacto envía un nuevo mensaje en menos de 2 horas y no hay un ticket pendiente, se reabre el último cerrado en lugar de crear uno nuevo, manteniendo el flujo "🕰️ Esperando", "👩‍💻 Atendiendo" y " Resuelto".
